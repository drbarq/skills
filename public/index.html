Co-branded Card Bonus Dispute

You opened the card for the welcome bonus, hit the spend, and the miles, points, or status-qualifying points (United PQP, Delta MQD…) never fully posted. This runs the whole recovery, end to end.

→Reconciles your card statements against the loyalty activity export to find exactly which piece is missing.
→Diagnoses the failure — duplicate account, status-points half dropped, partial post, wrong-account routing.
→Corroborates the offer you signed up for, including a Wayback snapshot of the issuer's own page.
→Generates the packet — timeline PDF, status doc, ready-to-send messages for both sides, phone scripts, tracker.
Load it

In Claude Code or Cowork, paste this. Claude fetches the catalog, finds this skill, and installs it:

Go to https://joetustin.com/skills/index.json, find the cobranded-card-bonus-dispute skill, download its tar.gz, and install it into my Claude skills folder (~/.claude/skills/).

Prefer to do it yourself? One line drops the skill into your personal skills folder:

mkdir -p ~/.claude/skills && curl -fsSL https://joetustin.com/skills/cobranded-card-bonus-dispute.tar.gz | tar -xz -C ~/.claude/skills/

No terminal needed. Download the skill file, then upload it in Settings → Capabilities → Skills (paid plans):

How loading works
01 — Point

Send Claude here

Tell Claude to read the catalog at joetustin.com/skills/index.json — a plain list of every skill and where to fetch it.

02 — Find

Match the skill

Describe what you need — "the United/Chase bonus one" — and Claude picks the matching entry by its keywords.

03 — Install

Drop it in

Claude downloads the skill and unpacks it into your skills folder. Next time it's relevant, Claude just uses it.

Self-install needs a Claude that can reach the web and your filesystem — Claude Code or Cowork. On Claude.ai, use the Tell Claude link to grab the file, then upload it under Settings. Exact menu paths shift as the app evolves; support.claude.com has the current steps.
